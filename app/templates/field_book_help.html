Field-Book Setup

Install Field Book from the Play Store. Search for "Field Book" by WheatGenetics. Once installed you will need to place the "fields" and "traits" files (see general help page) into the appropriate directories. This can be achieved by many methods, including by downloading the files directly from this site onto your device then using your device file manager. The Field Book app also provides a facility to use the "Dropbox" service to import these files.

  • Place the ".csv" file into Field-Book's field import directory ("fieldBook/field_import").
  • Place the ".trt" file into Field-Book's trait directory ("fieldBook/trait").

Data collection

  • Launch the app and in the menu set person and location.
  • Select Fields => Add (plus symbol) => Local => Select your ".csv"
    • As Unique identifier select UID
    • As primary and secondary order select categories that suit the order you will collect data from these items
  • Select Traits => List (three dots at top right) => Import/Export => Import => Select your block.trt or tree.trt as appropriate
  • Once these are imported you can switch between traits without losing data recorded from these traits.
  • Be careful to select the appropriate traits before collecting data (blocks vs trees).

Note that each unique identifier coming from the "fields.csv" file can not be duplicated. If you wish to update your fields (e.g. consolidating multiple files into one) make sure to upload all collected data to this site first (see below), as all data collected against those unique identifiers will be removed from your device. Then you may delete the old fields file and replace it with the new custom version. Alternatively, if you are adding a new set of trees you may simply wish to leave these in a separate fields file.

Exporting data from Field Book

From the Field Book app click the icon in the top left => select "Settings" => then "Export".

  • Under "Format" check the box for "Database"
  • Under "Field columns" select "Only unique identifier" (either option will work)
  • Under "Traits" select "All traits"
  • Then click "OK"

Depending on your device this may prompt you to share the file, at which point you may save it locally or even email it to yourself.


Submit data


Select the submission type ("Field Book (.csv)") then proceed as described on the general help page.

Note, a maximum of 64 traits may be exported at once. If you need to collect information on more than 64 traits then this may be exported as multiple files by selecting the active traits in Field Book
