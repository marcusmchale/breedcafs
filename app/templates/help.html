Field-book

To assist field data collection and ensure standardised reporting, the BreedCAFS database tools support use of the Field Book app for Android mobile devices.

To collect data with Field-Book you need two files, one that describes the trees (fields.csv), and another that describes the traits to be measured (traits.trt). Tools to generate both of these files are provided on this site.

Registering fields and trees

Visit Create => Register Trees.
Here you can add new Countries, Regions, Farms and Plots to the database. Trees can then be registered and will be assigned unique ID's in the database. A fields.csv file will be generated and a copy will emailed to your user address. This fields.csv file is to be placed in Field-Book's field_import directory. Subsequently, custom fields.csv files for any given plot can be generated for a range of supplied TreeIDs.

You can see the number of trees registered within each location in the tree chart provided.

Selecting traits

Visit Create => Create traits.trt.
Here you can select from a range of traits for which you may be collecting data. At the bottom of the list is a button labeled "Create". Click this to have a traits.trt file generated for the selected traits and emailed to your registered address.

Uploading data from Field Book

From the Field Book app click the icon in the top left => select "Settings" => then "Export".
- Under "Format" check the box for "Database"
- Under "Field columns" select "Only unique identifier"
- Under "Traits" select "All traits"
- Then click "OK"

Note:
- a maximum of 64 traits may be exported at once. If you need to collect information on more than 64 traits then this may be exported as multiple files by selecting the active traits in Field Book.
